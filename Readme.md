### TryCatch
```sh
use Throwable;

try {
  # code ...
} catch (Throwable $e) {
   $response = [
       'message' => $e->getMessage(),
       'file' => $e->getFile(),
       'line' => $e->getLine(),
       'Details' => $e->getTrace(),
   ];
}
```
### Date diff for debugs
```php
$start = new \DateTime;
$end = new \DateTime;
$calc = $end->diff($start);
echo $calc->format("%H:%I:%S");
```
